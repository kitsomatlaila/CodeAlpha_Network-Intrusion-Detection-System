Active Detection Rules

SQL Injection Detection
Active

Detects common SQL injection patterns in HTTP requests

alert tcp any any -> any 80 (msg:"SQL Injection Detected"; content:"union"; content:"select"; nocase; classtype:web-application-attack; sid:1000001; rev:1;)
Port Scan Detection
Active

Detects horizontal and vertical port scanning activities

alert ip any any -> any any (msg:"Port Scan Detected"; detection_filter: track by_src, count 5, seconds 10; classtype:attempted-recon; sid:1000002; rev:1;)
Malware Download Attempt
Active

Flags known malicious file downloads and patterns

alert tcp any any -> any any (msg:"Known Malware Download"; content:"|0f 00 00 00 0f 00 00 00|"; content:"exe"; classtype:trojan-activity; sid:1000003; rev:1;)
